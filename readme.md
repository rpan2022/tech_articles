# 文章管理

编程技术类文章先在 draft 文件夹中
排版使用 [markdown2wx](https://quaily.com/tools/markdown-to-wx/)
公众号文章联动知乎
这个仓库作为视频脚本仓库
后期可能考虑 AI 视频生成

## 工作流总结

1. 微信公众号
   md file -> 排版[markdown-to-wx](https://quaily.com/tools/markdown-to-wx/) -> 发表

2. 知乎
   自动导入微信公众号文章，已关联

3. 小红书
   md file -> 排版[md2card](https://md2card.cn/zh/editor) -> 发表
